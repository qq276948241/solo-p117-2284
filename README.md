# 2D 打飞机

Pygame 实现的 2D 太空射击游戏。

## 操作

- 方向键 ←→ 或 A/D：左右移动飞船
- 空格键：发射子弹

## 玩法

- 击落从天而降的陨石得分，大陨石分裂成中陨石，中陨石分裂成小陨石
- 陨石被击毁时会产生碎片飞溅的粒子爆炸效果
- 时间越久陨石下落越快

## 道具

- 红色星星：子弹升级为三连发，持续 10 秒
- 蓝色星星：获得护盾，持续 8 秒。护盾存在时撞到陨石不会死亡，护盾代替飞船承受一次撞击后消失

## 难度

开始菜单可选择简单 / 普通 / 困难，影响陨石数量和速度。

## 分数

历史最高分保存在本地 `high_score.json`。
